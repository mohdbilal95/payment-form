## Payment Form

### Description

This Payment Form is a simple and user-friendly web application that allows users to make payments using multiple payment methods. Customers can enter their payment details and choose their preferred payment option, such as Indian Rupees (₹), debit cards, credit cards, Visa, Mastercard, UPI, or net banking.

The form is designed to provide a smooth and secure payment experience, making it easy for users to complete transactions online.

### Features

* Easy-to-use payment interface
* Support for Indian Rupees (₹)
* Support for debit and credit cards
* Compatible with Visa and Mastercard
* UPI payment option
* Net banking support
* Responsive design for mobile and desktop devices
* Basic form validation

### Supported Payment Methods

* Cash (₹ INR)
* UPI
* Debit Cards
* Credit Cards
* Visa
* Mastercard
* Net Banking

### Use Cases

* HTML

### Installation

1. Clone the repository:

   ```bash
   git clone <repository-url>
   ```

2. Open the project folder:

   ```bash
   cd payment-form
   ```

3. Open `index.html` in your browser or run the project using a local server.
